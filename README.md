Docker-PostgreSQL
</h2>

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/hwJames/Docker-PostgreSQL.git
   git pull
   ```

2. Edit Setting
   ```sh
   vi .env
   vi docker-compose.yml
   ```

3. Docker Compose Start
   ```sh
   docker-compose up
   ```
